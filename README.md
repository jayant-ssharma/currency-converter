A real-time currency converter built with HTML, CSS, and JavaScript.

## Features
- Converts between 30+ world currencies
- Live exchange rates via API
- Auto-updates flag on currency change
- Swap currencies with arrow button

## Built With
- HTML, CSS, JavaScript
- [Fawazahmed0 Currency API](https://github.com/fawazahmed0/exchange-api)
- [Font Awesome](https://fontawesome.com) for icons
- [Flags API](https://flagsapi.com) for country flags

## How to Use
1. Select source and target currency
2. Enter amount
3. Click Convert

## What I Learned
- `async/await` and `fetch` API
- Error handling with `try/catch`
- DOM manipulation
- Working with external APIs
