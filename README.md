## What It Does
Convert between 150+ world currencies in real time using live exchange rates. Select your currencies, enter an amount, and instantly see the converted value.

---

## Features
- 🌍 150+ currencies supported
- 🔄 Swap currencies with one click
- 🏳️ Country flags for visual clarity
- ⚡ Live exchange rates via API

---

## Tech Used
- HTML · CSS · JavaScript · ExchangeRate API

## What I Learned

- DOM manipulation based on API responses
- Building a clean, functional UI with vanilla CSS
